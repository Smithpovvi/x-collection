## Installation project

node version 16.17.0

pm - yarn

```bash
$ yarn
```

## Running the app

```bash
# development watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# e2e tests
$ yarn run tests:e2e

# unit tests
$ yarn run tests:unit
$ yarn run test:unit-watch
```
